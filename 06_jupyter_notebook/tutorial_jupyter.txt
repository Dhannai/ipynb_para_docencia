 
#####TUTORIAL JUPYTER NOTEBOOK


 ###Instalar Jupyter Notebook y Python:
  
 Uno de los requisitos para instalar Jupyter Notebbok y su funcionamiento, es contar previamente con alguna versión de Python instalada.
 
 En caso de no tener Python,se recomienda instalar Anaconda, el cual consiste en un distribuidor que instala en forma secuencial Python,
 Jupyter Notebook y otros paquetes usados para computación cientifica. Es posible acceder a las instrucciones a traves del siguiente link:

 https://www.continuum.io/downloads

 En él veremos los procedimientos para instalar Anaconda en 3 casos, según el sistema operativo usado:

  * Windows
  * Mac OS-X
  * Linux 
 
 En primer lugar se debe descargar el paquete para instalar Anaconda, el cual presenta 2 opciones según la versión de Python que se 
 desee instalar:
 
 *versión Python 2.7
 *versión Python 3.5

 Una vez realizado lo anterior en la términal se debe copiar la siguiente insturcción, según el sistema operativo:
 
 * Windows:  hacer doble-click en el archivo.exe descargado y seguir las instrucciones 
 * Mac OS-X: hacer doble.click en el archivo.pkg descargado y seguir las instrucciones
 * Linux:    copiar en la términal los siguientes comandos:
              
               -bash Anaconda2-2.5.0-Linux-x86_64.sh (versión Python 2.7)
               -bash Anaconda3-2.5.0-Linux-x86_64.sh (versión Python 3.5)

               (el comando bash debe ir independiente si la términal trabaja con Bash-Shell)

 Otro punto a tener en cuenta es si se desea instalar la versión de 32-bit o 64-bit, que depenederá de las características del equipo.
 Una forma alternativa de realizar lo anterior (para usuarios de Linux) es la siguiente:
 
 * instalar pip mediante el siguiente comando en la términal: 

   -sudo apt-git install pip
   -sudo apt-get install build-essential python-dev
  
    (pip es un sistema de gestión usado para instalar y administrar paquetes de softwares escritos en Python)
 
 * ya hecho lo anterior, pip nos permitirá instalar los paquetes necesarios para Jupyter Notebook: 

   -pip install upgrade
   -pip install jupyter



 ###Correr Jupyter Notebook:

 En caso de tener Windows o Mac se puede correr haciendo doble-click en el programa
 En el caso de Linux, este se puede hacer desde la términal, direccionando la carpeta en la que tenemos el contenido de Jupyter
 y luego escribir el comando:
  
  -Jupyter Notebook

 Se abrirá una ventana en la que podremos ver los archivos posibles de correr con Jupyter, estando conectado a internet con el navergador Firefox 
 y accediendo a los archivos haciendo doble click.



 ### Editor de Texto Markdown

 Jupyter utiliza por defecto un editor de texto conocido como Markdown, que es un lenguaje de marcado ligero buscando tener máxima legibilidad
 y facilidad de publicación tanto en la entrada como la salida.
 
 Es por esto, que aquel que deseé trabajar con Jupyter Notebook debe tener conocimiento básico de la sintaxis ocupada por Markdown. A continuación
 presentamos una lista de opciones disponibles por este editor de texto:
 
 1) Tipos de encabezados

   # tipo1
   ## tipo2
   ### tipo3
   #### tipo4
   ##### tipo5

 2) Texto con énfasis

   * texto-cursiva *
   ** texto-negrita **

 3) Texto tipo codigo
  
   `texto-codigo`
  
   ```
     texto-codigo
     programa
   ```

 4) Tipos de listado
   
   * listado punteo
   1. listado enumerado

 5) Enlaces a otras direcciones
 
   [Texto del enlace aquí](URL "Título del enlace")

 6) Insertar imagines

   ![Texto alternativo](URL "Título de la imagen")
